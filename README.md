# training_stats [![Build Status](https://travis-ci.org/vinntreus/training_stats.svg?branch=master)](https://travis-ci.org/vinntreus/training_stats)

## Setup on local dev
```shell
make env
source settings.sh
```

## Start web
With gunicorn
```shell
make web
```
With flask development server
```shell
python app.py
```
